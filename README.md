# FinSight 💰

Personal finance dashboard with AI-powered spending insights.

Built to apply enterprise finance domain knowledge from my internship
(ERP Finance & Sales systems) to a consumer-facing product.

## Tech Stack

**Backend:** Java 21, Spring Boot 3, Spring Security (JWT), Spring Data JPA
**Frontend:** React 18, Recharts
**Database:** PostgreSQL
**AI:** Auto-categorization & monthly spending insights via LLM API
**DevOps:** GitHub Actions CI, Docker

## Planned Features

- [ ] User registration & JWT authentication
- [ ] Add / edit / delete transactions
- [ ] Category management
- [ ] Dashboard with monthly spending charts
- [ ] AI auto-categorization of expenses
- [ ] AI monthly insight summaries
- [ ] Budget goals & alerts
